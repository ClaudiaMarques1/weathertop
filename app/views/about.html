About

This is a combination of data, technology and code to provide a knowledgable application for users

This Weather Application is for users to be able to sign up and enter the reading of the weather in their local area.
This application will let you know by the wind speed if it is a light breeze to thunder depending on the readings you add for each Station.
This application has been dedicated to making living lives safer, easier and better with giving accurate prediction. As a technical application, it strives to utilise the latest technological advances in order to improve the efficiency, effectiveness and accuracy of its forecasts.
